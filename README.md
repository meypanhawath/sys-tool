# Sys-Tool

A simple Linux system management tool (runs with sudo).  

## Features
- System information report (CPU, RAM, storage, top processes, listening ports)
- Firewall control (UFW: enable, disable, allow/deny ports)
- SSH remote helper

## Installation

Run this command:

```bash
curl -sSL https://raw.githubusercontent.com/meypanhawath/sys-tool/master/install.sh | sudo bash
